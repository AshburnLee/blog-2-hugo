3. LLM decode 的过程是什么？

*答*：就是生成下一个 Token。这包含了解码策略，即如何选择下一个 Token，greedy search，beam search 等。然后，选择了一个 token 后，它会被添加到输入序列中，然后 LLM 再次运行，预测下一个 token。这个过程会一直重复，直到生成了结束 token，或者达到了预设的最大长度。【方法是在 decode 时，指明 `max_new_tokens=200`】
