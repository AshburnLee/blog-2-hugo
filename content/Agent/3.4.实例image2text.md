大多数 HuggingFace 上的“image-to-text”模型可直接通过 transformers pipeline 调用

~~~py
from transformers import pipeline
pipe = pipeline("image-to-text", model="JackChew/Qwen2-VL-2B-OCR")
result = pipe("your_image.jpg")
print(result)
~~~
